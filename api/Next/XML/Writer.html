Prints/Returns the XML Output Memory

Prints/Returns the XML Output Memory

If a Response Object is set, a 'Content-type' header will be sent and the output printed.

Otherwise the output will be returned 'as is'

Parameters

Response $response Response Object If set, a Content-Type Header will also be sent
boolean|optional $decode When returning, defines whether or not the entities will be decoded

Return Value

Response|string|void If a Response Object has not been provided then the http://www.php.net/XMLWriter Memory Buffer will be returned as string

Otherwise the Memory Buffer will be flushed, with or without, the proper Header. However, if the response Object provided has been configured to be returned instead of flushed — through Response::shouldReturn() — nothing is returned
