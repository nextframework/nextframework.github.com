Object shift(boolean $shift = FALSE)

Shifts the first element off the Collection, regardless of its index within it, therefore it's not affected by Collection::shuffle(), for example

Parameters

boolean $shift If TRUE the first element will be really removed from the Collection before being returned, along with its metadata.

If FALSE (default) it will be just retrieved, without affecting Collection's data

Return Value

Object If the Collection has any elements, its first Object will be returned. Otherwise, nothing is returned

Exceptions

UnderflowException Thrown if trying to shift an Object from an empty Collection
