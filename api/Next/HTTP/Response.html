Response addHeader(mixed $header, string|optional $value = NULL, float $schemaVersion = 1.1)

Add response Header to be sent

This method intermediates Next\HTTP\Headers\AbstractHeaders::addHeader() in order to use the proper field for HTTP Response Status Code Headers

Parameters

mixed $header Header name. Possible values are:

  • An integer as HTTP Response State Header

    In this case a Raw Header Field will be sent

  • A well-formed Header Field, instance of \Next\HTTP\Headers\Field

    In this case it'll be used "as is"

  • A Header Field Name

    We'll try to match a valid one or send it as "Generic"

  • An array of Header Fields, where each value can be a well-formed Header Field or a Header Field Name

    In this case we'll add them recursively
string|optional $value Header value
float $schemaVersion Schema Version. Only used if $header is an integer
