at line 253
protected init()

Additional Initialization

If only one argument is provided, it'll be checked against \Next\HTTP\Stream\Adapter\Adapter.

If it passes, it will be used and Request URI will be retrieved from it

Otherwise, this argument will be THE Request URI

The second argument is the Request Method, also optional.

If no arguments are entered (case of Routed Requests), default values will take place: REQUEST_URI from $_SERVER and GET as Request Method
