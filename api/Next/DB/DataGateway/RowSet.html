RowSet

class RowSet extends Object implements DataGateway, Iterator, ArrayAccess

The DataGateway RowSet Class creates an Entity instance for each result in a ResultSet filling their fields with Table Column values

Also, being a DataGateway, the resulting Entities can be updated or deleted after being modified
