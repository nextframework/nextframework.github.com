RowSet findBy(array $conditions, array $order = [], string|array|optional $limit = NULL)

Finds Entities in the Repository that match a set of criteria, optionally limiting the number of records on the resulting RowSet and with a custom ordering Clause

# Find data by User ID — PRIMARY KEY `user` — restricted by
# IDs '1' and '2' in descendant order

$manager -> findBy(
    [ [ 'user = :user', [ 'user' => [ 1, 2 ] ] ] ], [ 'user' => Query::ORDER_DESCENDING ]
);

Parameters

array $conditions One or more conditions to condition the Entity
array $order One or more fields to order the Entity in the resultset
string|array|optional $limit Restrictions on how many Entities will be present in the resultset and from which offset the finding process will start
