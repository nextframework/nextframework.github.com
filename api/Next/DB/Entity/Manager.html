Manager

class Manager extends Object

The Entity Manager Class is responsible for listing, updating and removing records based on informations provided by its associated Entity through a Repository

It also bridges its own Context to Query Builder's and the provided Repository's Contexts as well as the inverse, bridging the Repository to both its and Query Builder's Contexts
