Response

class Response extends Object implements Schema

Defines a Caching Schema through HTTP Response Headers, sending Content-Type, Content-Length, Last-Modified, Cache-Control and E-Tag Header Fields for resources that are allowed to be cached.

It's also responsible to send 304 and 404 Headers when given resource is, respectively, already cached or not found
